04 — Playback Controls

Adds Play / Pause / Reset buttons under the timer, so you can run it straight from vMix or OBS — enable mouse clicks on the browser source/input and you're set. Only available for a duration countdown or a count-up; a fixed date or a plain clock can't be paused. Crop the buttons out of frame if you don't want them on stream.

The play/pause state is remembered by the browser it's running in, so reloading the browser source in vMix/OBS resumes right where you left off — no need to reopen the builder.

05 — Style

12 ready-made looks. Pick one to skip styling entirely, or use it as a starting point below.

06 — Size

Text is sized relative to the browser source's own resolution, not fixed pixels — so instead of stretching the vMix/OBS layer bigger (which blurs), set the browser source's Width/Height to your target size and use this to fine-tune. It never gets pixelated because it's re-rendered at that size, not scaled up from a smaller image.

07 — Custom CSS optional

Overrides the template above. Target: .timer-wrap, .segment, .value, .label, .sep, .date-line, .ended-text.

How to use it

To avoid it looking pixelated when large: set the browser source's own Width/Height to the size you actually need (e.g. full canvas size), rather than adding it small and then stretching it bigger in the layer. Use the Size section above to fine-tune from there.

OBS Studio

  1. Sources → + → Browser
  2. Paste the URL above
  3. Set Width/Height (e.g. 1920×1080)
  4. Check "Shutdown source when not visible" off, so it keeps time

vMix

  1. Add Input → Web Browser
  2. Paste the URL above
  3. Set the resolution to match your output
  4. Overlay it on your program layer
